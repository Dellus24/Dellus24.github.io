HOW TO ADD A PROJECT
====================

1. Duplicate the "project-name" folder and rename it to your project
   (use lowercase with dashes, e.g. "101-gates" or "mir-archive")

2. Open meta.md and fill in all the fields

3. Drop your image files into the images/ subfolder

4. Tell Claude to sync — it will update the site automatically


CATEGORY OPTIONS
----------------
  academic      shows under Academic Work in the sidebar
  employment    shows under Employment in the sidebar


NOTES
-----
- The folder name becomes the project ID on the site (keep it short, no spaces)
- Images can be .jpg, .png, or .webp
- If you have a 3D model (.glb file), put it in the images/ folder and add its filename to the model: line
- To hide a project temporarily, rename its folder to start with _ (e.g. _101-gates)
